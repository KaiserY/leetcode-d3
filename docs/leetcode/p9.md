# 9. Palindrome Number

## Example

```
Input: x = 12321
Output: true
Explanation: 12321 reads as 12321 from left to right and from right to left.
```

<P9 />

## Rust Solution

```rust
impl Solution {
    pub fn is_palindrome(mut x: i32) -> bool {
        let mut y = 0;

        if x < 0 || (x % 10 == 0 && x != 0) {
            return false;
        }

        while x > y {
            y = 10 * y + x % 10;
            x /= 10
        }

        x == y || y / 10 == x
    }
}
```
